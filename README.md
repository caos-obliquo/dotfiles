# dotfiles

Monochrome Hyprland rice.

## Stack
hyprland  waybar  wofi  kitty  dunst  nvim
## Install
```bash
git clone https://github.com/caos-obliquo/dotfiles.git ~/dotfiles
cd ~/dotfiles/stow_home && stow */
